# NSquared.Ng.Utils

Some useful directive of filter for angularjs.

## Usage

1. Install the required tools: `gulp`, `bower`, `tsd`, `typescript`, `karma-cli`

		npm install -g gulp bower tsd@next typescript karma-cli

1. Install dependency

		npm install
		bower install
		tsd install
	or

		tsd install angular
		tsd install jquery

1. Build 

		gulp

1. Test

		gulp test
